# Research Log

研究内容や実験記録を効率的に管理・検索するために開発したWebアプリケーションです。

日々の研究では、実験条件や考察を継続的に記録する必要があります。しかし、一般的なメモアプリでは研究内容をタグで整理したり、目的の記録を素早く検索したりすることが難しいと感じました。

そこで、**React・Spring Boot・PostgreSQL** を用いて、研究ログを作成・管理・検索できるWebアプリケーションを開発しました。

## 工夫した点

### 1. タグ管理機能

ログには複数のタグを設定でき、タグをクリックすることで関連する研究ログを素早く検索できます。

また、ログを削除した際には未使用となったタグも自動で削除するように実装し、タグ一覧が不要なタグで増え続けないように工夫しました。

### 2. 検索機能

キーワード検索とタグ検索を独立した状態として管理し、両者を組み合わせた複合検索を実装しました。

検索処理にはSpring Data JPAのJPQLを利用し、タイトル・本文検索とタグ検索を柔軟に組み合わせられるよう設計しています。

### 3. 保守性を意識した設計

Controller・Service・Repositoryに責務を分離し、データベース操作や業務ロジックを適切に分割しました。

また、Reactではコンポーネントを分割し、フォーム・一覧・タグ表示を独立したコンポーネントとして実装しています。

## 起動方法

### Frontend

```bash
cd frontend
npm install
npm run dev
```

### Backend

```bash
./mvnw spring-boot:run
```

### Database

PostgreSQLを起動し、`application.properties` の設定に合わせてデータベースを作成してください。
